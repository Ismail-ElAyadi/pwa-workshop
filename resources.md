## Icons

To generate a complete set of icons for every type of devices -> https://favicomatic.com/

Check "every size" and complete "URL", "Title" and "Background color" in advanced settings .

Then copy the code that has been generated (or go get it in the zip you downloaded) and include it in index.html, in head.

Put the icons in your chosen file and check the path is correct in your code.
